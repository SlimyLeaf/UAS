PROSEDUR PENDAFTARAN SIDANG PROJECT

  1. Waktu pendaftaran Sidang Project tanggal 21-31 Juli 2021
  2. Pelaksanaan Sidang Project secara Daring tanggal:
    1. 4 Agustus 2021(Kelas Reguler A)
    2. 7 Agustus 2021(Kelas Reguler B/Karyawan)
  3. Isi form pendaftaran pada link Pendaftaran di atas
  4. Unggah semua hasil scan berkas persyaratan (jadikan 1 file pdf max 10mb berurutan) pada link pendaftaran dengan nama file PRJ_TAHUN_NIM_NAMA.
  5. Unggah Soft Copy (satu file office word) Laporan Project yang sudah disetujui dosen pembimbing pada link pendaftaran
  6. Hard Copy berkas persyaratan sidang PKL dimasukan dalam 1 map (Warna Ungu untuk prodi TI dan Warna Putih untuk Prodi SI) diserahkan ke staf BAAK pada saat pengumpulan hardcover laporan
  7. Harap memperhatikan aturan sidang:
    1. Memakai Jas Almameter pada sidang.
    2. Masuk meeting room tepat waktu
